주석은 #define, 구조체, 전역 변수, 주요 함수 를 중점으로 작성하였습니다. 한글/영어가 다소 섞여있습니다.
주요함수는 다음과 같습니다.
	- static void append_process (Process *process) // 프로세스 추가함수
	- static void simulate (int sched)	// process scheduling algorithm 시뮬레이션 하는 함수
	- int main (int argc, char **argv)  // main 함수
	
본 프로그램은 우분투 리눅스 18.04.1 LTS 환경에서 개발 및 실행되었습니다.
